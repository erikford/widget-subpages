== Description ==

A very simple widget that will list all sub pages of the currently viewed page. The widget will also list the sub pages when viewing a sub page.

== Installation ==

You can install Sub Pages Widget by uploading the extracted `widget-subpages` folder to your `/wp-content/plugins/` directory. Once the plugin has been successfully installed, simply activate the plugin through the Plugins menu in your WordPress Dashboard.
